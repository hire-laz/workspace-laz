# Huashu Design Skill — When to Use

Installed: 2026-04-22

## What It Does

- **Interactive Prototypes** — iPhone mockups, clickable web apps (10-15 min)
- **Slide Decks** — HTML + editable PPTX with real text frames (15-25 min)
- **Motion Design** — MP4/GIF animations with audio (8-12 min)
- **Infographics** — Magazine-grade data visualizations (10 min)
- **Design Critique** — 5-dimension expert review + actionable punch list (3 min)
- **Design Advisor** — 3 differentiated directions from 5 design schools (5 min)

## When to Trigger

Use huashu-design when Cho asks for:
- **"Build a prototype for [app]"** → Interactive prototype
- **"Make presentation slides with [topic]"** → HTML deck + PPTX export
- **"Create a promo animation for [product]"** → Motion design (MP4/GIF)
- **"Design an infographic showing [data]"** → Data visualization
- **"Review this design from multiple angles"** → 5D expert critique
- **"Give me 3 design directions for [brief]"** → Design advisor mode
- **"Build a clickable version of [wireframe]"** → Interactive prototype

## Key Features

- **Brand Protocol** — Auto-fetches logos, colors, fonts, product images from official sources
- **Multi-agent Fallback** — If brief is vague, auto-generates 3 differentiated directions
- **Real Exports** — PPTX are actual editable documents, not images; MP4/GIF are production-quality
- **Playwright-verified** — Click tests all interactive prototypes before delivery
- **Mobile Accurate** — iPhone 15 Pro bezel, Dynamic Island, real status bars

## Integration with My Work

Huashu-design works best for:
1. **Kaizen marketing materials** — slides, promos, social graphics
2. **The Lazy Lifter brand visuals** — landing pages, presentations, motion content
3. **NutriKaizen visuals** — infographics, testimonial layouts, process diagrams
4. **Quick design iterations** — when Cho needs multiple options fast

## How It Works

Triggers like a normal agent skill — just mention the design task. The skill:
1. Reads the brief
2. Extracts brand assets if a company/brand is mentioned (brand protocol)
3. Generates the design in parallel (e.g., 3 prototypes at once)
4. Exports ready-to-use files (HTML, MP4, PPTX, PDF)
5. Playwright tests interactive protos
